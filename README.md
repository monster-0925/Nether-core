# Nether-core

Nether is a Bitcoin Core fork with custom network parameters. Currently under development with planned ICO-based funding and future network launch.

## 🚧 Status
This project is under active development and not ready for production use.

## 🧱 Features
- Based on Bitcoin Core
- Custom genesis block
- Modified network parameters
- Experimental wallet support

## 🛣 Roadmap
- [ ] Stable testnet
- [ ] Multi-node synchronization
- [ ] Wallet improvements
- [ ] Public network launch
- [ ] Experimental token mechanics and ecosystem features

## 💡 Vision
Nether aims to explore new network configurations and funding mechanisms, including potential IPC-based models.

In the long term, the project will experiment with:
- Simplified token creation mechanisms (inspired by platforms like pump-style launch systems)
- DeFi-oriented primitives and asset models
- Cryptographic puzzle-based challenges and rewards
- A self-sustaining on-chain economy built around Nether
